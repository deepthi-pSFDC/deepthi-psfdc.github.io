## About

A colorful, single-page portfolio built with pure HTML & CSS — no frameworks, no dependencies, no build step. Hosted for free on GitHub Pages.

- **About** — Background and personal highlights
- **Skills** — Categorized skill tags (Product Management, Agile, AI, Tools, Core Competencies)

## Deploying Updates

1. Edit `index.html`
2. Commit and push to the `main` branch
3. GitHub Pages auto-deploys within ~1 minute
